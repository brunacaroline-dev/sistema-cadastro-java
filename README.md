# Sistema de Cadastro em Java

Projeto desenvolvido para praticar lógica de programação em Java.

## Funcionalidades
- Cadastro de usuários (nome e idade)
- Listagem de usuários
- Busca de usuário por nome

## Tecnologias
- Java

## Como executar
1. Compile:
   javac Cadastro.java

2. Execute:
   java Cadastro

## Objetivo
Projeto voltado para aprendizado em Análise e Desenvolvimento de Sistemas.
